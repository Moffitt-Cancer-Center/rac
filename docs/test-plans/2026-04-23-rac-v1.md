# RAC v1 — Human Test Plan

Operator-run verification of acceptance criteria that cannot be exercised without a real Azure subscription, Entra tenant, DNS authority, or GitHub Environments. Every AC without automated coverage is mapped to a numbered step here. Everything else is verified by 940 automated tests across 4 test suites (see the Automated Coverage section at the bottom).

Scope: **Operator-only**. Do not run in CI; this is a one-shot bootstrap + smoke test.

## Prerequisites

- Azure dev subscription; bootstrap service principal has Owner.
- Tier 1 bootstrap complete per `docs/runbooks/bootstrap.md` (management group, Log Analytics, Key Vault, storage account for state, GitHub OIDC federated credential).
- Environment variables set: `AZURE_LOCATION`, `PARENT_DOMAIN`.
- GitHub Environments `dev`, `staging`, `prod` configured with reviewer protection.
- Automated suites all green:
  - `cd apps/control-plane/backend && uv run pytest` → 652 passing
  - `cd apps/control-plane/frontend && pnpm test` → 84 passing
  - `cd apps/shim && uv run pytest` → 134 passing
  - `cd /home/sysop/rac-pipeline && uv run pytest` → 70 passing

## End-to-End Integration Scenario

Validate the full researcher journey across all 8 phases:

1. Submit `clean-python-flask` as a researcher in dev.
2. Pipeline builds and scans; `awaiting_research_review` within ~10 min.
3. Research approver approves → `awaiting_it_review`.
4. IT approver reviews manifest + findings; approves.
5. Provisioning runs; `deployed` with DNS + signing key + all 4 Tier 3 tags.
6. Mint reviewer token; open `https://<slug>.<PARENT_DOMAIN>/?rac_token=<jwt>` in incognito.
7. Interstitial loads (idle app); redirects to upstream within 20s; `access_log` row.
8. Revoke token; wait 60s; reload → revoked page.
9. Toggle `access_mode=public`; reload without token → 200; `access_log` with null jti.
10. Admin disables PI in Entra → overnight sweep flags the app → admin transfers to new PI → original `approval_event` rows unchanged.
11. Cost dashboard reflects MTD spend within 24h of first ingest.
